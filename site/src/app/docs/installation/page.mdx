# Installation

## Install the Library

<CodeGroup>

```bash {{ title: 'npm' }}
npm install uvcanvas
```

```bash {{ title: 'yarn' }}
yarn add uvcanvas
```

```bash {{ title: 'pnpm' }}
pnpm install uvcanvas
```

</CodeGroup>

```tsx
import { Lumiflex } from "uvcanvas";

export default function Home() {
  return (
    <div>
      <Lumiflex />
    </div>
  );
}
```
